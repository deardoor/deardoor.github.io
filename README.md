# DearDoor GitHub Pages

这是 DearDoor Studio 的静态站点仓库，用于承载应用官网与多语言隐私协议页面。

## 已上线应用
- I'm Smoke-Free（iOS）
  - 官网：/im-smoke-free/index.html
  - 隐私协议：/im-smoke-free/privacy.html
  - 图标：/im-smoke-free/app-icon-512.jpg
- Tomato-Mac（macOS）
  - 官网：/tomato/index.html
  - 隐私协议：/tomato/privacy.html

## 站点入口
- 主页：/index.html
- 广告声明：/app-ads.txt

## 目录结构
- im-smoke-free/
  - 官网与隐私协议、截图资源
- tomato/
  - 官网与隐私协议

## 本地查看
- 直接用浏览器打开对应 HTML 文件即可
